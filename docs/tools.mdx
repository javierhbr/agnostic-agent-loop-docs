## MCP vs. Direct File Access

<CalloutBox type="red" label="Never do this:">

```bash
# ❌ Agent reads spec files directly
cat openclaw-specs/constitution/policies.md
cat components/cart/invariants.yaml
```

This bypasses governance. Files can be incomplete, the agent doesn't get structured responses, and there's no audit trail.

</CalloutBox>

<CalloutBox type="teal" label="Always do this:">

```bash
# ✅ Agent calls MCP
platform_mcp.get_constitution()
component_mcp.get_invariants("cart")
component_mcp.check_invariant("CART-INV-001", proposal)
```

MCP validates the call, returns structured data, logs the access, and can return computed results like invariant violation checks.

</CalloutBox>
