## Autopilot: process backlog tasks sequentially

Preview without writing changes:

```bash
agentic-agent autopilot start --dry-run
```

Process up to N tasks:

```bash
agentic-agent autopilot start --max-iterations 5
```

## Work: guided claim -> build -> validate -> complete

```bash
agentic-agent work
```

TDD mode (creates RED/GREEN/REFACTOR subtasks and checks the TDD pack is installed):

```bash
agentic-agent work --task TASK-001 --follow-tdd
```

## Validate: keep drift out

```bash
agentic-agent validate
agentic-agent validate --format json
```
