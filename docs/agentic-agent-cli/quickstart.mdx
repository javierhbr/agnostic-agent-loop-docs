## 1) Install

Prereqs: Go 1.22+, Git.

```bash title="Install (recommended)"
go install github.com/javierhbr/agnostic-agent-loop@latest
agentic-agent version
```

## 2) Initialize a project

```bash title="Create a project"
mkdir my-project && cd my-project
agentic-agent init --name "My Project"
```

## 3) Create a track (idea -> spec)

```bash title="Track init (creates brainstorm scaffolding)"
agentic-agent track init "User Authentication" \
  --purpose "Secure user login" \
  --success "Users can register, login, logout"
```

Use the generated `brainstorm.md` with your AI agent, then validate spec completeness:

```bash title="Spec completeness check"
agentic-agent track refine user-authentication
```

## 4) Activate (plan -> tasks)

```bash title="Activate + decompose into tasks"
agentic-agent track activate user-authentication --decompose
```

## 5) Claim one task, generate context, validate

```bash title="Work a single task"
agentic-agent task claim TASK-001
agentic-agent context generate internal/auth
agentic-agent validate
agentic-agent task complete TASK-001
```
