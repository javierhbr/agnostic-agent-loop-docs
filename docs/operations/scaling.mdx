## Ready to start on Day 1?

Four phases: foundation, first initiative, standard flow, harden.

<div className={styles.d1Grid}>

<D1Phase
  label="Phase 1 · Foundation"
  items={[
    'Create openclaw-specs/ repo',
    'Write constitution/policies.md',
    'Deploy Platform MCP (reads specs/)',
    'Deploy Component MCP per component',
    'Symlink .claude/agents/ to specs/agents/',
  ]}
/>

<D1Phase
  label="Phase 2 · First Initiative"
  items={[
    'Select one low-risk pilot feature',
    'Run quick workflow (developer.md only)',
    'Developer calls Component MCP',
    'Verifier produces verify.md',
    'Validate gate checklist end to end',
  ]}
/>

<D1Phase
  label="Phase 3 · Standard Flow"
  items={[
    'Run standard workflow for medium feature',
    'Architect produces feature-spec.md',
    'Fan out developer agents per component in parallel',
    'Verifier waits for all component tracks',
    'Update graph/index.yaml post-merge',
  ]}
/>

<D1Phase
  label="Phase 4 · Harden"
  items={[
    'Enable full workflow for critical changes',
    'Activate Analyst for Guaripolo/live flag cases',
    'Require human approval in verifier.md',
    'Enforce verify.md sign-off in CI before merge',
  ]}
/>

</div>

<CalloutBox type="amber">
  The model scales by adding governance discipline, not by adding process services.
</CalloutBox>
